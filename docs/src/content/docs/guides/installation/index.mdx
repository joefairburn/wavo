## Package Installation

Install wavo using your preferred package manager:

```bash
# npm
npm install wavo

# pnpm
pnpm add wavo

# yarn
yarn add wavo

# bun
bun add wavo
```

## Requirements

- React 18 or higher
- TypeScript 5.0+ (optional but recommended)

## Framework Setup

### Next.js (App Router)

Wavo components use client-side features. Add the `"use client"` directive or use `client:load`:

```tsx
"use client";

import { Waveform } from "wavo";

export function AudioPlayer() {
  return (
    <Waveform dataPoints={audioData}>
      <Waveform.Bars />
    </Waveform>
  );
}
```

### Next.js (Pages Router)

Works out of the box. Import and use directly:

```tsx
import { Waveform } from "wavo";

export default function Page() {
  return (
    <Waveform dataPoints={audioData}>
      <Waveform.Bars />
    </Waveform>
  );
}
```

### Astro

Use the `client:load` directive for interactive components:

```astro
---
import AudioPlayer from "../components/AudioPlayer";
---

<AudioPlayer client:load />
```

### Vite / Create React App

Works out of the box. Import and use directly:

```tsx
import { Waveform } from "wavo";

function App() {
  return (
    <Waveform dataPoints={audioData}>
      <Waveform.Bars />
    </Waveform>
  );
}
```

## TypeScript

Types are included with the package. No additional `@types` package needed.

```tsx
import type { WaveformData, ProgressHandle } from "wavo";

const audioData: WaveformData = [0.1, 0.4, 0.8, 0.6, 0.3];
```

## Verify Installation

Test your installation with a basic waveform:

```tsx
import { Waveform } from "wavo";

const testData = [0.2, 0.5, 0.8, 0.6, 0.4, 0.7, 0.3, 0.9, 0.5, 0.2];

function TestWaveform() {
  return (
    <div style={{ height: 100, width: 300 }}>
      <Waveform dataPoints={testData}>
        <Waveform.Bars />
      </Waveform>
    </div>
  );
}
```

If you see a waveform visualization, you're ready to go.
