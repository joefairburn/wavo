## Understanding the Data Format

Waveform data is an array of normalized amplitude values between 0 and 1:

```tsx
import type { WaveformData } from "wavo";

// Each value represents amplitude (0 = silence, 1 = max)
const audioData: WaveformData = [0.1, 0.4, 0.8, 0.6, 0.3, 0.9, 0.5, 0.2, 0.7, 0.4];
```

The number of bars rendered is determined by the container width and bar dimensions, not the array length. Wavo automatically reduces larger datasets to fit.

## Adding Interactivity

Enable click and drag seeking:

```tsx
import { useState } from "react";
import { Waveform } from "wavo";

function InteractiveWaveform() {
  const [progress, setProgress] = useState(0);

  return (
    <Waveform
      dataPoints={audioData}
      progress={progress}
      onClick={(percentage) => setProgress(percentage)}
      onDrag={(percentage) => setProgress(percentage)}
    >
      <Waveform.Bars />
      <Waveform.Progress color="#f96706" />
    </Waveform>
  );
}
```

## Visualization Styles

### Bars (default)

Vertical bars with customizable width, gap, and radius:

```tsx
<Waveform dataPoints={audioData}>
  <Waveform.Bars width={3} gap={2} radius={2} />
</Waveform>
```

### Path / Line

Smooth curves or jagged lines:

```tsx
<Waveform dataPoints={audioData}>
  <Waveform.Path />
</Waveform>
```

## Compound Component Pattern

Wavo uses a compound component pattern. The Container provides context to child components:

```tsx
<Waveform dataPoints={audioData} progress={0.5}>
  <Waveform.Bars /> {/* Visualization */}
  <Waveform.Progress /> {/* Progress indicator */}
</Waveform>
```

Or use direct imports:

```tsx
import { Waveform, Bars, Progress } from "wavo";

<Waveform dataPoints={audioData}>
  <Bars />
  <Progress />
</Waveform>;
```
